JavaScript 匿名函数的直接调用

匿名函数的直接调用的主要目的是创建一个作用域边界，避免函数内部的变量和函数污染全局作用域。通过立即调用函数，函数内部的变量和函数都会在执行完后销毁，不会对全局作用域造成影响。
