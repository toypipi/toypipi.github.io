Echarts 图表中动态添加单位

发表于 2023-07-07 | 分类于 技术 |
字数统计: 517 | 阅读时长 ≈ 2

现在有一个需求是：前端根据用户勾选的状态参数绘制多条曲线。

实现思路：

  1. 前端获取到用户勾选的状态参数。
  2. 将状态参数列表发给后端。
  3. 后端将状态参数的数据发给前端。
  4. 前端根据后端发送的参数，分别构建 legend，X 轴，Y 轴数据，并生成图表需要的 option，更新图表。

参考文档：
Echarts 折线图数值后添加单位 提供了根据曲线名查找对应的单位思路。
Echarts 的 tooltip 中动态单位设置（使用 formatter 函数加工） 提供了保存 this 的思路。
