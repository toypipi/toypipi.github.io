Vue 响应式系统的工作原理

Vue  在初始化时会将  data  中的属性转换为  getter/setter.

当 Vue 实例创建时：

  • Vue 会遍历 data  中的所有属性
  • 使用 Object.defineProperty(Vue 2) 或 Proxy(Vue 3) 将这些属性转换为 getter/setter
  • 这些 getter/setter 允许 Vue 追踪依赖和更新视图
