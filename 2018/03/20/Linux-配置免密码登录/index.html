Linux 配置免密码登录

发表于 2018-03-20 | 更新于 2020-11-03 | 分类于 技术 | 评论数： | 阅读次数：

  1. 使用 ssh-keygen -t rsa 生成公钥，一路敲回车。

  2. 将生成的公钥发送到远程主机上

  3. 成功

注意
如果希望 ssh 公钥生效需满足至少下面两个条件：

  1. ssh 目录的权限必须是700
  2. ssh/authorized_keys 文件权限必须是600
